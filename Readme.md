## Blip Transport and Logistics System
A comprehensive platform that streamlines passenger transport and logistics operations. It enables bus booking and ticketing, parcel and luggage delivery management, consignment tracking, and payment processing. The system enhances efficiency with real-time updates, role-based access control, and detailed reporting, ensuring seamless coordination between operators, passengers, and logistics clients.

## Features
- Bus booking and ticketing
- Parcel and luggage delivery management
- Consignment tracking
- Payment processing
- Real-time updates
- Role-based access control
- Detailed reporting

## Technologies Used
- Java
- Spring Boot
- SQL (PostgreSQL)
- Maven
- Docker

## Setup Instructions
1. Clone the repository
2. Configure your database in `src/main/resources/application.yml`.
3. Build the project
4. Run the application

## Usage
- Access the web interface at `http://localhost:7071`
- Use provided endpoints for booking, tracking, and management
